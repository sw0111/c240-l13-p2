# ShoreSquad Project

## Overview
ShoreSquad is a web application designed to mobilize young people for beach cleanups. The app provides users with tools to rally their crew, track weather conditions, and plan eco-friendly activities with ease.

## Features
- **Crew Management**: Easily invite friends and organize beach cleanup events.
- **Weather Tracking**: Get real-time weather updates to plan your outings.
- **Interactive Map**: Locate nearby beaches and cleanup spots.
- **Social Features**: Share your experiences and connect with other eco-warriors.

## Color Palette
- **Ocean Blues**: Reflects the sea and promotes a calming effect.
- **Sandy Beiges**: Represents the beach and creates a warm, inviting atmosphere.
- **Vibrant Greens**: Symbolizes nature and eco-friendliness, encouraging environmental action.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/ShoreSquad.git
   ```
2. Navigate to the project directory:
   ```
   cd ShoreSquad
   ```
3. Open the project in your preferred code editor.
4. Install any necessary dependencies (if applicable).
5. Start the Live Server to view the application:
   ```
   npm start
   ```
